2.  [**`DCICFlix - Microservices Movie System`**](https://github.com/JRoman-brug/DCICFlix)
*A distributed movie recommendation platform built with a microservices architecture.*
     - **Architecture:** Specialized system composed of services: **Movies, Random Movies, Rating, Opinions, and Recommender**.
     - **Backend Stack:** Developed using **Node.js & Express**
     - **Data Persistence:** Utilizes **MongoDB** to handle movie catalogs and user ratings efficiently.
     - **Asynchronous Logic:** Implements event-driven communication using **RabbitMQ** to decouple the Rating and Opinions services.
     - **Infrastructure:** Fully containerized using **Docker** and orchestrated with **Docker Compose** for consistent environment deployment.
